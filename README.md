# Tetris
Tetris clone made with SDL2

### About
This project is a recreation of the game Tetris.

### Libraries used
**SDL 2.0** for windowing and rendering  

### Controls
A: Move left  
D: Move right  
S: Move down  
Left shift: Rotate  
Space: Hard drop  
F: Switch piece
